# UnderFy
um app voltado a usar a api do spotify para ranquear artistas menos conhecidos com base nos gostos do cliente

##objetivos 
- auxiliar o crescimento de novos artistas independentes
- ampliar os horizontes para ouvintes enjoados das mesmas músicas
